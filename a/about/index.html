项目简介

工作性价比计算器是一个现代化的Web应用，旨在帮助用户科学地评估和对比工作性价比。 通过收集用户的工作信息，运用科学的算法模型，为用户提供个性化的职业分析报告。

核心功能

  • 工作性价比计算：基于多维度指标的科学计算
  • 横向对比分析：与同行业、同地区、同年龄段用户对比
  • 数据可视化：直观的图表展示分析结果
  • 个性化报告：针对性的职业发展建议

技术栈

  • 前端框架：Nuxt 3 + Vue 3
  • 样式框架：Tailwind CSS
  • UI组件：Nuxt UI
  • 状态管理：Pinia
  • 国际化：Nuxt i18n
  • 构建模式：SSG静态站点生成

开发理念

我们相信，通过数据驱动的分析，可以帮助人们做出更明智的职业选择。 工作性价比不仅仅是薪资的体现，更是工作与生活平衡的综合考量。
